Bloa Documentation

Syntax

Bloa uses a simple, readable syntax inspired by modern scripting languages.

Variables

x = 42
name = "Bloa"

Expressions

result = (x + 10) * 2
is_even = x % 2 == 0

Control Flow

If/Else

if condition {
  // code
} else {
  // code
}

While Loop

while condition {
  // code
}

For-in Loop

for item in list {
  say item
}

Repeat Loop

repeat 5 {
  say "Hello"
}

Functions

function add(a, b) {
  return a + b
}

result = add(5, 3)

Classes and Objects

class Person {
  function __init__(self, name) {
    self.name = name
  }

  function greet(self) {
    say "Hello, " + self.name
  }
}

p = Person("Alice")
p.greet()

Inheritance

class Student extends Person {
  function study(self) {
    say self.name + " is studying"
  }
}

Modules

Use 'use' to import modules and 'require' to include files.

use "math"
result = math.sqrt(16)

require "utils.bloa"

Built-in Functions

  • say(value) - Print to console
  • ask(prompt) - Read input from user
  • len(collection) - Get length
  • str(value) - Convert to string
  • int(value) - Convert to integer
  • float(value) - Convert to float
  • range(start, end) - Create range
  • append(list, item) - Add item to list

Standard Library

The standard library includes modules for math, I/O, and strings.

Math

  • math.sin(x)
  • math.cos(x)
  • math.sqrt(x)

I/O

  • io.read_file(path)
  • io.write_file(path, content)

String

  • string.upper(s)
  • string.lower(s)
  • string.split(s, delimiter)
