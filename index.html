About Bloa

Bloa is a minimalist interpreter for a scripting language designed for ease of use and expressiveness. It supports variables, functions, classes, and more, making it perfect for scripting tasks.

Features

Variables & Expressions

Support for arithmetic, logic, and comparison operations.

Control Flow

if/else, while, for-in, and repeat loops.

Functions & Classes

Define functions and classes with inheritance.

Modules

Use 'use' for importing and 'require' for including files.

Built-in Functions

print, range, len, str, int, float, append, and more.

Standard Library

Math, I/O, and string utilities.

Code Examples

Hello World
say "Hello, World!"
Functions
function greet(name) {
say "Hello, " + name
}

greet("Bloa")
Classes
class Animal {
function speak(self) {
say "Animal sound"
}
}

class Dog extends Animal {
function speak(self) {
say "Woof!"
}
}

d = Dog()
d.speak()
Lists
list = [1, 2, 3]
for item in list {
say item
}
